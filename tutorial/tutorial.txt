When using this program, the file locations are set to their default locations and should work when put in the root directory. Next click on Choose Students. This will open a new window with a list of students to test. 

You can select one or all to test. Once you comfirm your amount, click Ok.

If you want to have your results on a window after it's done, leave the box checked. Click on Run to start. 

The program will start running and the bar will show the progression of the run. When finished, the output panel will appear.  

You can click which students' result to see on the left. You have three opitions from there.

If you want to edit the results, click on Open Output to edit on trivial edit. Then save the change through File.

If you want locate the file of a student, hightlight the student and click Open Source. The file should start with output. 

If you are satisfied, click on Close Window and then the close icon to close the program.
